Setup instructions:

Colabs are special notebooks with embedded code libraries and dedicated memory so you can both embed and run code as well as write text content. 
Open this Colab in either a code editor of your choice copy it to Google Drive (reccommended) to start running the code!


Note: the CSV (Comma, Seperated, Value) files can be named whatever you wish, make sure to update the file name in the notebook correctly as well.
